## Image Processing Workflow

### Step 1: PDF Image Extraction (Sougwen Chung)

**Tools & Methods:**

Option A: **Adobe Acrobat** (Recommended if available)
```
1. Open SOUGWEN CHUNG 2025 - CONGSHENG.pdf
2. Tools → Export PDF → Select "Image" format
3. Choose page(s): 2, 4, 5, 6
4. Output format: JPEG
5. Resolution: 600 DPI minimum
6. Color: RGB
```

Option B: **Online PDF Converter** (Free alternative)
```
1. Go to smallpdf.com or ilovepdf.com
2. Upload PDF file
3. Select "PDF to JPG" conversion
4. Set quality: Highest/Maximum
5. Download individual pages
```

Option C: **Python Script** (Fastest for batch)
```python
import pdf2image

# Convert PDF pages to images
pages = pdf2image.convert_from_path(
    "SOUGWEN CHUNG 2025 - CONGSHENG.pdf",
    dpi=600,
    first_page=2,
    last_page=6
)

for i, page in enumerate(pages):
    page.save(f"sougwen_page_{2+i}.jpg", "JPEG", quality=95)
```

### Step 2: Image Quality Verification

**For all 4 images:**

| Criterion | Target | Verification |
|-----------|--------|--------------|
| **Resolution** | ≥3000×2000px | Check file properties |
| **Color Space** | RGB | Verify in image editor |
| **File Format** | JPG | Not PNG/BMP (saves space) |
| **Compression** | Lossless quality | Use TinyPNG, not aggressive compression |
| **Aspect Ratio** | ~1.5:1 or 16:9 | Measure dimensions |
| **Color Accuracy** | True colors (no color cast) | Visual inspection |

### Step 3: Compression Pipeline

**Tool: TinyPNG (tinypng.com)**

```
1. Open tinypng.com
2. Drag & drop all 4 images
3. Download compressed versions
4. Target final size:
   - Sougwen images (Meridians): 1.8-2.0 MB each
   - Lauren images (AUTO): 1.9-2.2 MB each
5. Total for 4 images: ~7.8 MB (acceptable)
```

**Alternative: ImageMagick (Command line)**
```bash
convert input.jpg -quality 85 -strip output.jpg
# Strips metadata, reduces file size
```

**Tool: Squoosh or FFmpeg**

**Using Squoosh (Online):**
```
1. Go to squoosh.app
2. Upload each JPG
3. Select WebP format
4. Adjust quality to match JPG appearance
5. Download WebP versions
```

**Using FFmpeg (Command line):**
```bash
ffmpeg -i input.jpg -c:v libwebp -quality 80 output.webp
```

## Troubleshooting

**Problem: PDF export produces low-quality images**
- Solution: Use online converter with "Enhanced" quality setting
- Alternative: Contact artist for official image files

**Problem: Lauren McCarthy URLs return 404**
- Solution: Search get-lauren.net/Auto directly for alternative image links
- Alternative: Email artist at given contact

**Problem: TinyPNG shows files already optimized**
- Solution: Use ImageMagick with lossy compression (quality 82-85)
- Note: Don't sacrifice visual quality for file size

**Problem: QR code doesn't scan**
- Solution: Reduce margin, increase contrast between colors
- Test with multiple QR scanner apps
